# Gerenciador de Tarefas em Python

Um programa simples de gerenciamento de tarefas, feito em Python, que permite adicionar, visualizar, concluir e remover tarefas diretamente pelo terminal. O programa utiliza **cores ANSI** para destacar mensagens de sucesso, erro e informações do menu.

---

## Funcionalidades

- Adicionar tarefas com descrição personalizada.
- Visualizar todas as tarefas cadastradas com status **Pendente** ou **Concluída**.
- Marcar tarefas como concluídas.
- Remover tarefas da lista.
- Menu interativo com cores para facilitar a visualização.

---

## Cores usadas

- **Verde**: Mensagens de sucesso (adição ou conclusão de tarefas).  
- **Vermelho**: Mensagens de erro ou alerta (índice inválido, entrada inválida).  
- **Rosa**: Boas-vindas, menu e despedida.  
- **Azul**: Títulos de listas ou seções.
